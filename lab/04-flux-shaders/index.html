SIX FRAGMENT PROGRAMS,
COMPUTED AS YOU WATCH.

Every tile below is a hand-written GLSL fragment shader running in raw WebGL2 — no libraries, no textures, no video files. Hover a tile to overdrive its clock. Click one to inspect it at full resolution, source included.

SPECIMENS 006  ·  UNIFORMS u_time / u_mouse / u_resolution  ·  DEPENDENCIES 000

PROTOCOL

One shared animation clock drives every visible tile. Offscreen programs are halted by an intersection observer. Tiles render at reduced density; the full-resolution pass is reserved for the inspector.

INPUT

The pointer streams into every program as u_mouse, smoothed by exponential decay. Hovering a tile overdrives its local clock to ×2.2. Keys 1–6 jump straight to a program; ESC returns to the grid.

SYSTEM

Raw WebGL2 and hand-written GLSL ES 3.00. Zero libraries, zero textures, zero build step. If it is not arithmetic, it is not here.
